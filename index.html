Educational Data Mining

This was a part of final project in Artificial Intelligence class.
While we were given the task of choosing two datasets from UC-Irvine Machine Learning Repository, the requirements for the project were as follows:

  • [Dataset] Classification
  • [Dataset] Multivariate
  • [Dataset] # of Attribute(10 ~ 100)
  • [Dataset] # of Instances (100 ~ 1,000, Greater than 1,000)

  • [Analysis] Use at least 4 different classifiers for comparative analysis
  • [Analysis] The final evaluation must use cross-validation
  • [Analysis] Make sure to use the concept of overfitting in final evalution
  • [Analysis] Use the results of zeroR, oneR as the baseline
The two datasets used are as follows:
  • Student Performance dataset
  • Turkieye Student Evaluation
NOTE: If you would like to experiment with dataset that has been pre-processed to work with Weka, you can download it from my GitHub repository.
